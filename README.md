# -# Random Password Generator

## Автор: [Сомов Дмитрий Владиславович]
## Описание:
Простое приложение для генерации случайных паролей с настройкой параметров и сохранением истории.

## Возможности:
- Выбор длины пароля (6–32).
- Включение/отключение цифр, букв и спецсимволов.
- История паролей сохраняется в JSON.
- Интерфейс на Tkinter.

## Пример использования:
1. Запустите `main.py`.
2. Настройте параметры.
3. Нажмите «Сгенерировать».
4. Пароль появится в поле и в таблице истории.
